Files

All site files are divided into following folders.

css folder - Contains css files
js folder - Contains Javascript files
img & images folder - Contains all images
php folder - Contains all Contact Form related files.
fonts folder - Contains "Font-Awesome" fonts plugin

Headings

The heading is at the top of all home pages. There are 20 home pages in the version 1 of DOT and more to come in the later versions.
